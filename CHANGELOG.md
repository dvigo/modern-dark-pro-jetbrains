# Changelog

All notable changes to the "Modern Dark Pro for JetBrains" theme will be documented in this file.

## [1.0.0] - 2026-04-10

### Added
- Initial release of Modern Dark Pro theme for JetBrains IDEs
- Modern Dark Pro - Monokai variant with vibrant accent colors
- Modern Dark Pro - Night variant optimized for low-light environments
- Support for all JetBrains IDEs (IntelliJ IDEA, PyCharm, WebStorm, etc.)
- Comprehensive color palette for syntax highlighting
- High contrast design for excellent readability
- Optimized for extended coding sessions

### Features
- Two carefully crafted dark theme variants
- Professional design aesthetic
- WCAG AA compliant contrast ratios
- Support for all JetBrains IDEs 2020.1+
- Clean UI elements with modern styling

## Theme Color Specifications

### Modern Dark Pro - Monokai
- **Primary Background**: `#1e1e2e`
- **Primary Foreground**: `#d4d4d4`
- **Primary Blue**: `#58a6ff`
- **Success Green**: `#3fb950`
- **Warning Orange**: `#d29922`
- **Error Red**: `#f85149`
- **Purple Accent**: `#d2a8ff`

### Modern Dark Pro - Night
- **Primary Background**: `#0d1117`
- **Primary Foreground**: `#c9d1d9`
- **Subtle Blue**: `#1f6feb`
- **Muted Green**: `#238636`
- **Muted Orange**: `#8b7500`
- **Muted Red**: `#da3633`
- **Muted Purple**: `#6e40aa`
